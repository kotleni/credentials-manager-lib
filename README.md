## CredentialsStorageLib
Multi-platform Kotlin library for storing key-value credentials.

### Platforms
- [x] Kotlin/JVM
  - [x] macOS
  - [x] Linux
  - [x] Windows
  - [ ] Android
- [x] Kotlin/JS
    - [x] Browser
